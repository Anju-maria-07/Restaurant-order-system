# 🍽️ Restaurant Ordering System (Python)

## 📌 About the Project

The **Restaurant Ordering System** is a console-based Python application developed to simulate the order management process in a restaurant. It enables restaurant staff to create, manage, update, and bill customer orders while storing order data permanently using JSON.

This project was built to strengthen my understanding of Python programming, modular application design, file handling, JSON data persistence, exception handling, and Git/GitHub workflows as part of my journey toward becoming a Python/Django developer.

---

## ✨ Features

* 📖 Display Food Menu
* ➕ Create New Order
* 👀 View Current Orders
* 🍽️ Add Items to an Existing Order
* ❌ Remove Items from an Order
* 🗑️ Cancel an Order
* 💰 Generate Bill
* 🧾 Automatically Remove Completed Orders After Billing
* 💾 Store Orders Permanently Using JSON
* ⚠️ Handle Invalid User Input with Exception Handling
